# Retail Sales Performance Dashboard (Excel Data Analytics Project)

## Project Overview

This project is an end-to-end Retail Sales Analysis Dashboard built using Microsoft Excel. The goal of this project was to transform raw retail sales data into meaningful business insights through data cleaning, analysis, visualization, and dashboard creation.

The project simulates a real-world business scenario where retail management needs to analyze performance across sales, profit, customer segments, product categories, regions, and discount strategies.

This dashboard helps answer critical business questions such as:

* Which product categories generate the highest sales?
* Which categories are the most profitable?
* Which customer segments contribute the most revenue?
* Which regions perform best?
* How do discounts impact profitability?
* Which products are generating losses despite high sales?

---

## Project Objectives

The main objective of this project was to:

* Clean and prepare raw retail sales data
* Perform exploratory data analysis (EDA)
* Identify key business insights
* Create interactive dashboards using Excel
* Improve business decision-making through data visualization

---

## Key Insights from the Analysis

* Technology was identified as the highest-performing category in both sales and profit.
* Consumer segment contributed the highest overall sales.
* Central region showed the strongest business performance.
* Some product sub-categories generated high revenue but negative profitability.
* Discount strategies showed direct impact on profit margins.
